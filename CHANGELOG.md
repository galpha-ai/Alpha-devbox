# Changelog

All notable changes to Devbox Agent will be documented in this file.

## [0.1.0] - 2026-04-09

### Added

- Initial open-source release
- Two-process controller/runner architecture
- Docker and Kubernetes container runtimes
- Session management with persistent workspaces
- Multi-channel support: Telegram, Slack, Web (HTTP + WebSocket)
- Agent-as-Code definitions (CLAUDE.md + seed.yaml)
- Session garbage collection and lifecycle management
- Concurrency control and message queuing
- GitHub App integration for repo seeding
- Scheduled task execution (cron, interval, one-shot)
